This repository contains the analysis code for the results presented in: Claudia Gulea, Rosita Zakeri, Jennifer K. Quint : "Model-based comorbidity clusters in patients with heart failure: association with clinical outcomes and healthcare utilization"

Runs in R v.3.2 and requires the following libraries:
poLCA
dplyr
ggplot2
reshape2
Hmisc
nnet
mice
